# Rental CRM — краткий технический обзор

## Что это
Лёгкий одностраничный фронтенд (vanilla HTML/CSS/JS) для управления арендной техникой:
- техника,
- проекты,
- клиенты,
- транзакции,
- финансовая аналитика по окупаемости.

## Архитектура
- `index.html` — статическая разметка всех разделов.
- `styles.css` — единый файл стилей (dark theme, responsive).
- `app.js` — основной UI/state/API слой.
- `analytics.js` — дополнительная загрузка таблицы окупаемости (пересечение с логикой `app.js`).
